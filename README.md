  > Ministerio de Educación, Ciencia y Tecnología de la Nación & Programa Nacional Mapa Educativo (eds.) 2009. República Argentina – Pueblos Indígenas.
- the derived dataset using the DOI of the [particular released version](../../releases/) you were using

## Description


This dataset provides non-overlapping, contemporary geographic areas of indigenous populations in Argentina as published by the Ministry of Education of Argentina in 2009.

This dataset is licensed under a CC-BY-4.0 license
